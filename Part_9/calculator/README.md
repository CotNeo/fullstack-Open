# Calculator: Basic TypeScript Applications

This directory contains basic TypeScript applications and utilities.

## 📁 Structure

```
calculator/
├── bmi-calculator/      # BMI calculation application
├── exercise-calculator/ # Exercise tracking application
├── utilities/          # Utility functions and helpers
└── examples/           # Code examples
```

## 🎯 Learning Objectives

- CLI application development
- Input validation
- Error handling
- Module organization
- TypeScript best practices

## 📚 Applications

### BMI Calculator
- Health metric calculation
- Input validation
- Error handling
- Command-line interface

### Exercise Calculator
- Fitness tracking
- Data analysis
- Statistical calculations
- User feedback

### Utilities
- Common functions
- Type guards
- Validation helpers
- Error handling

## 🚀 Getting Started

```bash
cd calculator
npm install
npm run dev
```

## 📖 Features

- **Type Safety**: Full TypeScript coverage
- **Validation**: Input validation and sanitization
- **Error Handling**: Comprehensive error management
- **CLI Support**: Command-line interface
- **Modular Design**: Reusable components

## 🛠️ Usage

### BMI Calculator
```bash
npm run bmi 180 74
```

### Exercise Calculator
```bash
npm run exercises 2 1 0 2 4.5 0 3 1 0 4
```
